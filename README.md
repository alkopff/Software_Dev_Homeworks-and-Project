Files for CME 211 class called "Software Development for Scientists and Engineers" at Stanford University
Fall Quarter 2022

This repository contains the files for for all the homeworks and for the final project of the CME 211 class. 
The first part of the homeworks were written in Python (hw0 - hw4). The second part of the homeworks (hw5-hw6) were written in C++.

The project focuses on building a Conjugate Gradient Solver in C++, using sparse matrices operations.
